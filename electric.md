# 批量电子元件整理:
每个抽屉按 6x6 分割，横轴ABCDEF，纵轴123456，每个型号的芯片装入统一的密封袋，于是每个型号的芯片都有了独立坐标。<br>
入库同时用excel制作数据库文件，统计名称 封装 生产商 类型 数量 位置，便于查找，在使用过之后修改数量，可以统计余料，便于工程前规划购买。<br>

# 盘料整理
向大淘宝超市货柜车卖家定制盘料架，画好机械图一周交货，空料盘可以淘宝购买用于条状物品整理，电线软电缆使用下方洗车用绕线器整理，每个盘料也编入数据库，便于查找。<br>
连接器需要拍照整理

# 仪器工具整理

制定仪器工具命名规则：生产商+类型+产品ID（或自行编码），写入NFC芯片，在出房间时直接刷卡登记，贵重仪器使用远场射频芯片<br>
实验室白板（四个折叠支架，三米厚质角铝，两根工业导轨，（家用晾衣架）钢丝拉线器和滑轮）
